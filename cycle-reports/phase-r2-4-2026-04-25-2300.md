# Cycle 2 · Phase 4 (Public Site) · 2026-04-25 23:00

## Discovery
1. robots.txt didn't explicitly allow modern AI crawlers — GPTBot,
   ClaudeBot, PerplexityBot, Google-Extended treat ambiguous robots
   as opt-out, so our sport microsites + tournament pages were
   silently invisible to AI search and chat-driven discovery.
2. Sport microsites are long on mobile (hero → stats → pitch → 6
   program tiles → 4 FAQs → cross-link → bottom CTA) and the only
   visible CTA scrolls off for most of the read. Conversion leak.
3. Sport microsite metadata (canonical, OG image, twitter card,
   siteName) was already in place from a hook auto-format pass —
   no fixes needed there.

## Shipped
1. **AI crawlers explicitly allowed** in robots.ts: GPTBot, ChatGPT-User,
   ClaudeBot, Claude-Web, anthropic-ai, PerplexityBot, Perplexity-User,
   Google-Extended, Applebot-Extended, CCBot. Each gets the same
   admin/portal/api disallow list. Default `*` rule preserved.
2. **Mobile-only sticky inquire CTA** on every sport microsite —
   bottom-fixed, full-width, primary-CTA. Respects safe-area-inset-bottom
   for iPhone notch + home-indicator. Source-tagged `<sport>-stickymobile`
   so we can A/B against in-flow CTAs in analytics.
