## Gaps found this iteration

### P0 — fix this iteration
1. **/api/teams/logo POST IDOR** — `src/app/api/teams/logo/route.ts`. The route allows the `coach` role but never verified the coach actually owns the team being uploaded for. A coach could overwrite any team's logo. Added a join check against `teams.coachUserId` + `teams.name`.
2. **/api/admin/equipment/movement error message** — when a movement would push on-hand below zero the error message stitches in `msg.split(":")[1]` and would render `"current: undefined"` if the colon was missing. Added a fallback string.
3. **/api/subscribe Mailchimp DC parsing** — `apiKey.split("-").pop()` returns the whole key when there's no dash, producing a request to `https://<entire-key>.api.mailchimp.com/...` which always 404s but emits the email + key into the logs anyway. Now skips the request and logs a config-fix warning when the suffix is missing.
