## Gaps found this iteration

### P0 — fix this iteration
1. **CORS preflight reflected `Origin` header with `*` fallback** — `src/middleware.ts:78`. Today no `Access-Control-Allow-Credentials` is set so the wildcard isn't directly exploitable, but a future change that adds it would have shipped a session-hijack vector. Restricted to the request's own host or `NEXTAUTH_URL` host; otherwise omit the header entirely.
2. **/api/me/profile password change had no rate limit** — an attacker with a stolen session cookie could brute-force the current password endlessly (bcrypt slows but doesn't stop). Added a 5-attempts-per-15-minutes limit keyed on userId.
3. **Newsletter signup logged the email in plaintext when Mailchimp wasn't configured** — `src/app/api/subscribe/route.ts:79`. PII in logs. Replaced with a non-PII info log.
