# Cycle 1 · Phase 5 (Customer Acquisition & Marketing) · 2026-04-25 10:23

## Scope
SEO, content, email capture, retention, referral, social proof, analytics events, A/B infra, retargeting, local SEO, review generation.

## Discovery — gaps in current marketing surface

### P0 — attribution + funnel tracking
1. **`inquire_form_submit` event isn't fired anywhere.** The event is defined in analytics types but the InquiryForm doesn't call it on success. Funnel tracking is broken.
2. **UTM params aren't auto-captured** in inquiry submissions. Visitors arriving via `?utm_source=instagram&utm_campaign=summer-camps` lose attribution because the form only reads `source` from a default page-slug.
3. **No short-link / redirect system.** SMS campaigns and QR codes rely on shortened, trackable URLs. Right now any short URL has to live external (bit.ly, goo.gl) which leaks data + costs reliability.

### P0 — review generation
4. **No `/reviews` redirect page** to funnel happy customers to Google reviews. Front desk can text a link after a great session — without a clean redirect URL, it's a 70-character Google review URL that gets blocked / mistyped.

### P1 — capture
5. **EmailSignup is not on sport microsites.** Visitors not ready to inquire have no way to opt in for email updates.
6. **No exit-intent capture** on inquiry pages.

### P2
7. No A/B test infra (flag-driven variant on hero copy).
8. No "as seen in" / press logo strip anywhere.

## Chosen tasks

1. **Fire `inquire_form_submit`** in InquiryForm on success with metadata `{kind, source}`.
2. **Auto-capture UTM** params from URL on inquiry pages — fold `utm_source/utm_medium/utm_campaign` into `source`.
3. **Short-link / redirect system** — new table `short_links` + `/r/[slug]` route that 302s and logs the click. Admin UI deferred to next pass.
4. **`/leave-a-review` redirect page** — clean URL → Google review redirect. Simple text version for SMS share.
